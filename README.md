# on-fit
누구나 쉽게 동네에서 함께 운동할 사람을 찾고, 실력/취향/시간대에 맞게 즉시 모임을 개설하거나 참여할 수 있는 운동 커뮤니티 플랫폼

1. 서비스 개요
서비스명: 운동 번개 (가칭)

목표:

누구나 쉽게 동네에서 함께 운동할 사람을 찾고, 실력/취향/시간대에 맞게 즉시 모임을 개설하거나 참여할 수 있는 운동 커뮤니티 플랫폼.

2. 주요 대상
주로 20~40대 직장인 및 대학생 중심의 근거리 운동 참여자

운동을 꾸준히 하고 싶지만 함께할 사람이 없는 사람

실력·연령·지역 기준으로 필터링된 매칭을 원하는 이용자

3. 주요 기능
🏠 메인 페이지
모집글 리스트: 운동 종목, 지역, 날짜, 인원, 실력 기준 표시

필터링: 종목 / 지역 / 날짜 / 실력 / 모집 상태(모집중/마감)

추천 모임: 내가 팔로우한 유저의 게시글 또는 동일 지역 인기 모임

📄 상세 페이지
모집글 상세 정보 (종목, 시간, 장소, 필요 장비, 모집 인원, 레벨 등)

참여 버튼 → 클릭 시 채팅방 자동 생성

게시글 작성자 프로필 보기 / 팔로우 / 신고 기능

👤 마이 페이지
내가 참여한 번개 모임 기록

내가 작성한 게시글 / 받은 뱃지 목록

누적 참여 횟수, 취소 횟수, 노쇼 여부 표시

🧍 프로필 페이지 (또는 모달)
프로필 이미지, 닉네임, 연령, 대표 종목, 실력 등 표시

팔로우 / 팔로워 목록

최근 활동 기록 (참여 모임, 후기 등)

✏️ 게시글 생성 페이지
종목 선택, 날짜/시간, 장소, 모집 인원, 실력 기준 선택

간단한 소개글 입력

모집글 등록 시 자동으로 메인 리스트 반영

📅 달력 페이지
날짜별로 예정된 번개 현황 한눈에 보기

참여 중 / 주최 중 / 팔로우 중인 모임 강조 표시

4. 인증 및 사용자 시스템

GPS 기반 자동 탐지 또는 직접 설정
